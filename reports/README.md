# Reports

Folder ini berisi laporan teknis dan visualisasi hasil analisis.

## Struktur

```
reports/
├── figures/          ← Output visualisasi dari notebook (PNG)
│   ├── viz1_eda_overview.png
│   ├── viz2_top_hashtags.png
│   ├── viz3_correlation_heatmap.png
│   ├── viz4_ab_testing.png
│   └── viz5_explanatory.png
└── laporan_teknis.pdf  ← Laporan akhir (generate setelah jalankan notebook)
```

## Cara Generate

1. Jalankan semua notebook secara berurutan
2. Visualisasi otomatis tersimpan ke `reports/figures/`
